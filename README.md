# Lottery Number Generator

A simple browser-based lottery number generator.

## Games
- Powerball (5 numbers 1–69 + 1 number 1–26)
- Ohio Lucky for Life (5 numbers 1–48 + 1 number 1–18)

## Notes
- Numbers are randomly generated
- All numbers are unique per draw
- For entertainment purposes only

To use:
